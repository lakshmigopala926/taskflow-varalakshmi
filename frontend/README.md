# 🚀 TaskFlow – Full Stack Task Manager

## 📌 Overview

TaskFlow is a full-stack task management application built using **React (Frontend)** and **Go + Gin (Backend)** with SQLite database.

It allows users to:

* Create projects
* Add tasks
* Track task status
* Perform full CRUD operations

---

## 🛠️ Tech Stack

### Frontend

* React.js
* Fetch API
* Bootstrap (optional)

### Backend

* Go (Golang)
* Gin Framework
* SQLite

---

## ⚙️ Features

* ✅ Create / Delete Projects
* ✅ Add / Delete Tasks
* ✅ REST API Integration
* ✅ Clean UI
* ✅ Full CRUD functionality

## 🚀 How to Run Locally

### Backend

```bash
cd backend
go mod tidy
go run .
```

### Frontend

```bash
cd frontend
npm install
npm start
```

---

## 🌐 API Endpoints

* GET /health
* GET /projects
* POST /projects
* DELETE /projects/:id
* GET /projects/:id/tasks
* POST /projects/:id/tasks
* DELETE /tasks/:id
